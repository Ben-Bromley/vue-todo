Welcome to Vuetilities! This is a simple collection of web-based utility apps I built using Vue 3 and the composition API.

## Setup and Running Locally

```bash
$ git clone https://github.com/Ben-Bromley/vuetilities.git
$ cd vuetilities
$ npm install
$ npm run dev
```
